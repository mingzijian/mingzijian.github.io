CentOS查看硬件信息
-----------------------------

### 查看CPU信息
```shell
lscpu
```
https://www.linuxcool.com/lscpu

### 查看PCI总线信息
```shell
lspci
```

### 列出 USB 控制器和与 USB 控制器相连的设备的详细信息
```shell
lsusb
```

### 列出不同分区的概要信息、挂载点、已用的和可用的空间。
```shell
df -h
```

### 查看系统中使用的、闲置的和 RAM 的总体数量
```shell
free -h
```

### 列出多种硬件单元的详细或者概要的信息
```shell
lshw -short
```
