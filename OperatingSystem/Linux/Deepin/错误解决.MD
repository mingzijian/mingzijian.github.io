Deepin 错误解决
------------------------

### 软件安装
#### 文件覆盖错误
- 错误描述：
正试图覆盖...它同时被包含于软件包...在处理时有错误发生...
正试图覆盖 x.x，它同时被包含于软件包 XX，在处理时有错误发生：xxxxx.deb

- 错误原因:
该文件因为之前的某个deb的安装已经有内容了，安装与他相关文件时，需要覆盖该文件，但是没有成功。

执行如下命令强制覆盖安装错误的内容，
```shell
dpkg -i --force-overwrite /xxxxx.deb
```
    dpkg ： 
    即 package manager for Debian ，是 Debian 和基于 Debian 的系统中一个主要的包管理工具，可以用来安装、构建、卸载、管理 deb 格式的软件包。
    
    -i ：
    即 install，用户请求安装该软件包
    
    --force-overwrite ：
    强制覆盖
    
    xxxxx.deb ：
    是你需要安装的deb文件(绝对路径)
    
- 示例：
```shell
dpkg -i --force-overwrite /var/cache/apt/archives/deepin-desktop-schemas_3.13.9+c2-1+comsta_amd64.deb
```
