# 十神

十神是根据日主（自身）与其他干支的**五行生克 + 阴阳属性**划分的，逻辑如下：

1. **生我者（印星）**：阳生阴 / 阴生阳为 “正印”，阳生阳 / 阴生阴为 “偏印”；
2. **我生者（食伤）**：阳生阴 / 阴生阳为 “伤官”，阳生阳 / 阴生阴为 “食神”；
3. **克我者（官杀）**：阳克阴 / 阴克阳为 “正官”，阳克阳 / 阴克阴为 “七杀”；
4. **我克者（财星）**：阳克阴 / 阴克阳为 “正财”，阳克阳 / 阴克阴为 “偏财”；
5. **同我者（比劫）**：阳同阳 / 阴同阴为 “比肩”，阳同阴 / 阴同阳为 “劫财”。



### **一、生我者为印星（生日主的五行）**

印星代表庇护、学识、思维、长辈（父母、师长）等，分为正印和偏印。

- **正印**
  - 含义：正统的学识、稳重的思维、善良慈悲、有责任感、重视传统与规则，象征长辈的关爱、正式的文凭、稳定的庇护（如工作单位、家庭）。
  - 特点：性格偏向温和、保守，擅长思考但不张扬，重视精神层面的满足。
- **偏印（枭神）**
  - 含义：非正统的智慧、独特的思维、灵感与创造力，也带有多疑、敏感、孤僻的特质，象征偏门知识（如玄学、艺术）、意外的帮助或短暂的庇护。
  - 特点：想法新颖但可能脱离常规，容易钻牛角尖，对神秘事物或小众领域感兴趣。

### **二、我生者为食伤（日主所生的五行）**

食伤代表才华、创造力、欲望、晚辈（子女）、自由意志等，分为食神和伤官（注：您提供的信息中未提及伤官，此处补充完整）。

- **食神**
  - 含义：温和的才华、乐观开朗、衣食无忧、善于享受生活，象征通过正当能力获得的福气（如才艺、美食）、孝顺的子女、平稳的创造力。
  - 特点：性格随和、知足常乐，能将才华自然展现，人际关系和谐。
- **伤官**
  - 含义：旺盛的创造力、叛逆精神、直言不讳，象征突破性的才华（如艺术、技术革新）、强烈的个人意志，但也可能过于自我、不服管束。
  - 特点：思维活跃、敢于挑战传统，适合从事自由职业或创新领域。

### **三、克我者为官杀（克制日主的五行）**

官杀代表约束、压力、事业、权力、伴侣（异性）等，分为正官和七杀（偏官）。

- **正官**
  - 含义：正统的约束、责任感、规则意识，象征正式的事业（如公职、稳定职业）、合法的权力、稳重的伴侣，也代表压力带来的成长。
  - 特点：性格正直、自律性强，重视社会规范，适合管理或体制内工作。
- **七杀（偏官）**
  - 含义：突发的压力、竞争挑战、偏门的权力，象征不稳定的事业（如经商、风险行业）、激烈的竞争、强势的伴侣，也代表爆发力与魄力。
  - 特点：敢闯敢拼、抗压能力强，但容易冲动或遭遇意外挑战。

### **四、我克者为财星（日主所克的五行）**

财星代表财富、物质、欲望、妻子（男性）、父亲（部分理论）等，分为正财和偏财。

- **正财**
  - 含义：稳定的财富、正当的收入（如工资、主业）、踏实的物质观，象征通过努力积累的资产、传统的家庭观念。
  - 特点：理财谨慎、重视实际利益，对金钱规划性强。
- **偏财**
  - 含义：意外的财富、灵活的收入（如投资、副业、奖金）、乐观的物质观，象征机会性的收益、不拘小节的消费态度。
  - 特点：擅长把握商机、花钱较大方，但财富可能波动较大。

### **五、同我者为比劫（与日主五行相同的五行）**

- **比肩**
  - 含义：同性同辈（如同性兄弟姐妹、朋友），象征平等的竞争、合作、自我意识，也代表稳健的助力或分享（如共同创业、朋友互助）。
  - 特点：性格独立、有自我主张，重视同辈情谊，但也可能因 “同我” 而产生利益分配的竞争。
- **劫财**
  - 含义：异性同辈（如异性兄弟姐妹、合作伙伴），象征带有争夺性的助力或损耗，可能是意外的帮助，也可能是破财、争夺资源的风险（如合作中被 “分走” 利益）。
  - 特点：行动力强、讲义气，但容易冲动，在财富或感情上可能有 “争夺” 倾向。
